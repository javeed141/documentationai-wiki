### Getting Started

<Callout kind="success" collapsed="false">
  **You're on the latest version:**

  API v4 is live with improved query performance and expanded property data coverage. [See what's new](/docs/api-introduction)
</Callout>

<Callout kind="tip" collapsed="false">
  **Authentication required:**

  All API requests require a valid Bearer token. <a href="https://portal.datafiniti.co/" target="_blank" rel="noopener noreferrer">Get your API key</a> from the Datafiniti portal to begin making requests.
</Callout>

<Callout kind="alert" collapsed="false">
  **Rate limits apply:**

  Free plans are limited to 1,000 records/month. Upgrade to a paid plan for higher limits and bulk access.
</Callout>
