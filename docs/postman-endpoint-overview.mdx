title: "Postman: Endpoint Overview"
---

This walkthrough will briefly describe each endpoint in the Datafiniti API.

## 1. Open Your Datafiniti Collection

When you click on your Datafiniti API v4 collection, you will have two subfolders. One with Public Endpoints and one with Secured Endpoints.

<Image src="https://files.readme.io/78b7db4-Screen_Shot_2017-12-13_at_1.42.34_PM.jpg" width="1920" height="1080" alt="Datafiniti API v4 collection showing Public and Secured Endpoints subfolders" noCaption />

## 2. Health Check

The "Health Check" endpoint can be used to confirm that your computer is successfully communicating with our API. Note the `200 OK` response code.

<Image src="https://files.readme.io/1ad2205-Screen_Shot_2017-12-13_at_1.42.49_PM.jpg" width="1920" height="1080" alt="Health Check endpoint showing 200 OK response" noCaption />

## 3. Authentication

"Authentication" will grab your user token when you enter in the e-mail address and password associated with your account.

<Image src="https://files.readme.io/3dc80b1-Screen_Shot_2017-12-13_at_1.42.58_PM.jpg" width="1920" height="1080" alt="Authentication endpoint for retrieving user token" noCaption />

## 4. Creating a View

When you use "Create A View," simply give it a `name`, `data_type`, and then set up the `fields` you'd like to see in that view. This view will be saved on your account for you to use when downloading records.

<Image src="https://files.readme.io/77cdb52-Screen_Shot_2017-12-13_at_1.48.11_PM.jpg" width="1920" height="1080" alt="Create A View endpoint with name, data_type, and fields configuration" noCaption />

## 5. Get All Views

"Get All Views" will generate a list of all views available on your account as well as what those views show.

<Image src="https://files.readme.io/6fb7689-Screen_Shot_2017-12-13_at_1.49.22_PM.jpg" width="1920" height="1080" alt="Get All Views endpoint showing list of available views" noCaption />

## 6. Get View By Name

If you know which view you are looking for and want to double check that it collects a specific value, you can use "Get View By Name" to pull up that view as well as all relevant information about it. You simply enter the name of the view you'd like to update at the end of the `GET` request.

<Image src="https://files.readme.io/d0b2dd3-Screen_Shot_2017-12-13_at_1.49.16_PM.jpg" width="1920" height="1080" alt="Get View By Name endpoint showing view details" noCaption />

## 7. Update View

"Update View" will allow you to change the fields, `name`, and `data_type` of a specific view. You simply enter the name of the view you'd like to update at the end of the `PUT` request.

<Image src="https://files.readme.io/bc0c795-Screen_Shot_2017-12-13_at_1.49.47_PM.jpg" width="1920" height="1080" alt="Update View endpoint for modifying view configuration" noCaption />

## 8. Delete View

"Delete View" will delete a view based on the `name` you put at the end of the `DELETE` request.

<Image src="https://files.readme.io/fec47f8-Screen_Shot_2017-12-13_at_1.50.08_PM.jpg" width="1920" height="1080" alt="Delete View endpoint in Postman" noCaption />

## 9. Get All Downloads

"Get All Downloads" will generate a list of all the downloads you have run on your account.

<Image src="https://files.readme.io/6f70daa-Screen_Shot_2017-12-13_at_1.50.21_PM.jpg" width="1920" height="1080" alt="Get All Downloads endpoint showing download history" noCaption />

<Callout kind="alert">
**Result File Expiration**

Keep in mind that your result files will expire after 7 days, even if the result file link shows up when you recall your downloads.
</Callout>

## 10. Get Download

"Get Download" is used to call a specific download result file. Simply enter the `download ID` at the end of the `GET` request. This request also displays relevant information pertaining to the download request such as if it's still running, when it was finished, number of records requested, data type, format, and the query used to generate the results.

<Image src="https://files.readme.io/22fbf1b-Screen_Shot_2017-12-13_at_1.50.52_PM.jpg" width="1920" height="1080" alt="Get Download endpoint showing download details and status" noCaption />

## 11. Cancel Download

<Image src="https://files.readme.io/91de654-Screen_Shot_2017-12-13_at_1.51.02_PM.jpg" width="1920" height="1080" alt="Cancel Download endpoint in Postman" noCaption />
