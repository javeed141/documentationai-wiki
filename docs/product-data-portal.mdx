## Product Data from the Portal

Here you'll find a step-by-step walkthroughs for maximizing Datafiniti's Product Data and API to fulfill your data needs. Let's get started!

## Navigating to Product Data

Once you log into your account, this will bring you to the Datafiniti Dashboard. From there you can use the left Navigation Bar to select searching different data types. In this case select "Product Data"

<Image src="https://files.readme.io/a4c519e06733a9ab38f6e46fb8f363d8901e9ea55392af564ca107a4b0983023-image.png" width="1920" height="1080" alt="Screenshot showing navigating to product data" noCaption />

Once you click "Product Data" you will be brought to Datafiniti's Product Data "All Filters"

<Image src="https://files.readme.io/0d165bb95986238eca7f7bdbcbe9972915e359b856d00da57a714245ab233c6e-image.png" width="1920" height="1080" alt="Screenshot showing navigating to product data" noCaption />

## Product Data Filter Builder

The Datafiniti Filter Builder is a GUI that allow you to compose API queries will relative ease. These filter fields are based off of our Datafiniti Product schema. For more information on what each filter field you can refer [here](/docs/product-data-schema). You can use the following steps to add a filter to your query.

## Product Data API Builder

You can also select the `API` tab to build a Product search query similar to the our API reference guide would allow you to build. You can reference the [API reference guide](/docs/constructing-product-queries) on how to build a query. In most cases you can copy and paste your API search query into the Product Data API Builder and click `Search`

<Image src="https://files.readme.io/a49f4997ca34632b2244106eddbdc77ec4045017520daa4186faafeddd2d6e6b-image.png" width="1920" height="1080" alt="Screenshot showing product data api builder" noCaption />

You can click on any box of a product within your search results to expand the record.

<Image src="https://files.readme.io/b93cf5ad577a6039eba66637ce6d3c9991e753e6a221e6bfe82bf949c4b79161-image.png" width="1920" height="1080" alt="Screenshot showing product data api builder" noCaption />

<Callout kind="info">
  **Credit Usage**

Please note that whenever you expand the record a credit will be charged to the account. For more on [How Credits Work](/docs/how-credits-work-1).
</Callout>

For help and tip on more complex querying techniques, you can find out more [here](/docs/advanced-querying).
