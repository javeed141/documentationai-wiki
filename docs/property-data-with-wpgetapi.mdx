---
title: "Property Data with WPGetAPI"
description: "Learn how to integrate Datafiniti's Property Data API into your WordPress site using the WPGetAPI plugin."
---

For this guide, we'll use the WPGetAPI plugin to access Datafiniti's property data API from your WordPress site.

**Your environment and data needs:**

- You're working with WordPress
- You want to display property data on your website

<Steps>
  <Step title="Install WPGetAPI plugin" icon="download" title-type="h3">
    1. In your WordPress admin, go to **Plugins > Add New**
    2. Search for "WPGetAPI"
    3. Click **Install Now** and then **Activate**
  </Step>

  <Step title="Get your API token" icon="key" title-type="h3">
    To get your API token:
    1. Go to the [Datafiniti Web Portal](https://portal.datafiniti.co/settings/api)
    2. Login and copy your API token
  </Step>

  <Step title="Configure the API" icon="settings" title-type="h3">
    1. Go to **WPGetAPI > Setup** in your WordPress admin
    2. Click **Add New API**
    3. Configure the following:
       - **API Name**: Datafiniti
       - **Base URL**: `https://api.datafiniti.co/v4`
    4. Save your settings
  </Step>

  <Step title="Add authentication" icon="lock" title-type="h3">
    1. Go to the **Headers** tab for your Datafiniti API
    2. Add a new header:
       - **Header Name**: Authorization
       - **Header Value**: `Bearer AAAXXXXXXXXXXXX` (replace with your token)
    3. Add another header:
       - **Header Name**: Content-Type
       - **Header Value**: `application/json`
  </Step>

  <Step title="Create an endpoint" icon="link" title-type="h3">
    1. Go to the **Endpoints** tab
    2. Add a new endpoint:
       - **Endpoint Name**: Search Properties
       - **Method**: POST
       - **Endpoint**: `/properties/search`
    3. In the **Body** section, add:

    ```json
    {
      "query": "country:US",
      "num_records": 10,
      "format": "JSON"
    }
    ```
  </Step>

  <Step title="Display data on your site" icon="eye" title-type="h3">
    Use the WPGetAPI shortcode to display property data:

    ```
    [wpgetapi_endpoint api_id="datafiniti" endpoint_id="search_properties"]
    ```

    Or use PHP in your theme:

    ```php
    <?php
    $properties = wpgetapi_endpoint(
        'datafiniti',
        'search_properties',
        ['debug' => false]
    );

    if ($properties && isset($properties['records'])) {
        foreach ($properties['records'] as $property) {
            echo '<div class="property">';
            echo '<h3>' . esc_html($property['address']) . '</h3>';
            echo '<p>' . esc_html($property['city']) . ', ' . esc_html($property['province']) . '</p>';
            echo '<p>Bedrooms: ' . esc_html($property['numBedroom']) . '</p>';
            echo '</div>';
        }
    }
    ?>
    ```
  </Step>
</Steps>

<Callout kind="tip">
  For more advanced WordPress integrations, consider using the PHP examples from our [PHP guide](/docs/property-data-with-php) in a custom plugin.
</Callout>
