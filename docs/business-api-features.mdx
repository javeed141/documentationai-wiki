title: API Features
---

Datafiniti’s Business Data API provides robust tools for tools for discovering, analyzing, and leveraging active business data, contact information, and state public records. With flexible query capabilities and extensive data coverage, you can build highly specific searches tailored to your use case. Listed here are several features Datafiniti's business search API allows you to use for people data searches:

<Columns cols="3">
  <Card title="Count" href="/docs/business-count" icon="calculator" horizontal="false">
    Retrieve the total number of business records matching your query criteria.
  </Card>

  <Card title="Geolocation" href="/docs/business-geolocation" icon="map-pin" horizontal="false">
    Search and filter businesses based on geographic coordinates and radius.
  </Card>

  <Card title="Pagination" href="/docs/business-pagination" icon="chevrons-right" horizontal="false">
    Navigate through large business data result sets using paginated responses.
  </Card>
</Columns>
