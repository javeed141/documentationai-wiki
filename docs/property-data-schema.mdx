title: "Property Data Schema"
---

The property data schema below lists all available fields in a property record. Note that actual property records (actually, most of them) may not have every field present.

<ParamField path="absenteeOwner" show-location="false" param-type="object" required="false" deprecated="false">
The owner of a property who does not reside in or otherwise occupy that property.

- `absent`: Is currently an absentee owner (boolean)
- `firstDateSeen`: Earliest date seen of the absentee owner. (date)
- `lastDateSeen`: Latest date seen of the absentee owner. (date)

**Example:**
`"absenteeOwner": [ { "absent": false, "firstDateSeen": "2024-04-11T17:17:53.270Z", "lastDateSeen": "2024-04-11T17:17:53.270Z" } ]`
</ParamField>

<ParamField path="address" show-location="false" param-type="string" required="false" deprecated="false">
The physical street address for this property.

**Example:**
`"address": "7206 Walling Ln"`
</ParamField>

<ParamField path="appliances" show-location="false" param-type="string" required="false" deprecated="false">
The appliances included with the property.

**Example:**
`"appliances": ["Cooktop","Dryer","Dishwasher","Microwave","Refrigerator","Washer"]`
</ParamField>

<ParamField path="architecturalStyles" show-location="false" param-type="string" required="false" deprecated="false">
The architectural style of the property.

**Example:**
`"architecturalStyles": ["Ranch","Bungalow"]`
</ParamField>

<ParamField path="assessedValues" show-location="false" param-type="object" required="false" deprecated="false">
The assessed value of the property. Each assessed value may contain:

- `improvementsAmount`: assessed value for improvements made to the property. (float)
- `landAmount`: assessed value for the land/lot of the property. (float)
- `totalAmount`: amount value the property is currently assessed for. (float)
- `year`: year value when the property was assessed. (integer)

**Example:**
`"assessedValues": [ { "totalAmount": 1745400.0, "improvementsAmount": 1745400.0, "landAmount": 1745400.0, "year": 1985 } ]`
</ParamField>

<ParamField path="brandTenant" show-location="false" param-type="string" required="false" deprecated="false">
The branded tenant that is currently associated with the property.

**Example:**
`"brandTenant": "Bob Evans Restaurants, LLC"`
</ParamField>

<ParamField path="brokers" show-location="false" param-type="object" required="false" deprecated="false">
A list of brokers that have represented this property over time. Each broker may contain:

- `agent`: The name of the real estate agent representing this property. (text)
- `company`: The company this agent works for (or the company representing this property). (text)
- `dateSeen`: The date this agent/company was seen representing the property. (date)
- `emails`: A list of email addresses associated with the agent or company. (keyword)
- `licenseNumber`: The license number of the agent representing the property. (text)
- `licenseType`: The license Type of the agent representing the property. (text)
- `people_key`: The corresponding key that link to Datafiniti's people data record. (contact, demographics, & more) (text)
- `phones`: A phone number associated with the agent or company. (keyword)
- `websites`: A list of websites associated with the agent or company. (keyword)
- `workEmail`: Email address specifically associated with the business the broker works for currently. (keyword)

**Example:**
`"brokers": { "emails": ["tomatterson@realtracs.com"], "licenseNumber": "1059280", ... }`
</ParamField>

<ParamField path="brokerCoop" show-location="false" param-type="boolean" required="false" deprecated="false">

**Example:**

**Example:**

A list of price estimates for the property.

- `currency`: The currency listed for amountMin and amountMax (string)
- `dateSeen`: The date when this estimatedPrice was seen for this property. (date)
- `pricePerSquareFoot`: The price divide by the square footage. (float)

**Example:**
`"estimatedPrices": [ { "amountMax": 295000, "amountMin": 295000, "currency": "USD", ... } ]`
</ParamField>

<ParamField path="features" show-location="false" param-type="object" required="false" deprecated="false">
A list of features associated with this property. The feature list can be collected from specific feature sections available on listings for this property. It can also serve as a catch-all field for miscellaneous characteristics of this property that don't make sense to include as top-level fields. Each feature contains:

- `key`: A label for this feature. (text)
- `replace`: Flag indicating value field is replaced versus appended. (boolean)
- `value`: A list of possible values for this feature. (text)

**Example:**
`"features": [ { "key": "Community District", "replace": "true", "value": [ "318" ] } ]`
</ParamField>

<ParamField path="fees" show-location="false" param-type="string" required="false" deprecated="false">
A list of various fees that have been seen for this property. Each fee will contain:

- `amountMax`: The maximum fee value listed. (float)
- `amountMin`: The minimum fee value listed. (float)
- `currency`: The currency listed for amountMin and amountMax (keyword)
- `dateSeen`: A list of dates when this fee was seen. (date)
- `type`: A classification value for the fee. (text)

**Example:**
`"fees": [ { "amountMax": 11, "amountMin": 11, "currency": "USD", ... } ]`
</ParamField>

<ParamField path="floorPlans" show-location="false" param-type="object" required="false" deprecated="false">

**Example:**

</ParamField>
