## Feature overview

| Feature | Description | Use case |
|---------|-------------|----------|
| Autotrace | Returns source URLs for data fields | Data verification, compliance |
| Count | Returns only the count of matching records | Estimate dataset size |
| Geolocation | Search by lat/long coordinates | Location-based searches |
| Pagination | Navigate large result sets | Process data in batches |

## Quick examples

### Get record count

```json
{
  "query": "province:TX AND mostRecentStatus:\"For Sale\"",
  "num_records": 0
}
```

Response includes `num_found` with total matching records.

### Search by location

```json
{
  "query": "geoLocation:[30.2,-97.8 TO 30.4,-97.6]",
  "num_records": 100
}
```

### Paginate results

```json
{
  "query": "province:TX",
  "num_records": 100,
  "scroll_token": "previous_scroll_token_here"
}
```
