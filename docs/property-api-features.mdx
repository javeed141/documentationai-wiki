---
title: API Features
---

Datafiniti's Property Data API provides robust tools for discovering, analyzing, and leveraging real estate information across residential, commercial, and public records. With flexible query capabilities and extensive data coverage, you can build highly specific searches tailored to your use case. Listed here are several features Datafiniti's API allows you to use for property data searches:

<Columns cols="2">
  <Card title="Autotrace" href="/docs/autotrace-property" icon="scan-search" horizontal="false">
    Automatically trace and track property data changes across multiple sources over time.
  </Card>

  <Card title="Count" href="/docs/count-property" icon="calculator" horizontal="false">
    Retrieve the total number of property records matching your query criteria.
  </Card>

  <Card title="Geolocation" href="/docs/geolocation-property" icon="map-pin" horizontal="false">
    Search and filter properties based on geographic coordinates and radius.
  </Card>

  <Card title="Pagination" href="/docs/pagination-property" icon="chevrons-right" horizontal="false">
    Navigate through large property data result sets using paginated responses.
  </Card>
</Columns>
