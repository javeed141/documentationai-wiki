title: Product Data Schema
---

The product data schema below lists all available fields in a product record.  Note that actual product records (actually, most of them) may not have every field present.

| Field Name | Example Value (in JSON) | Description |
| --- | --- | --- |
| `asins` | `"asins":  "B0009XCFRE"` | The ASIN (Amazon identifier) used for this product.   <br /> Field type: keyword |
| `brand` | `"brand": "Sumsung Electronics"` | The brand name of this product as presented by the data source.   <br /> Field type: text |
| `categories` | `"categories": [ <br />   "Home Improvement", <br />   "Heating, Cooling, & Air Quality", <br />   "Heaters", <br />   "Propane Heaters" <br /> ]` | A list of category keywords used for this product across multiple sources.   <br /> Field type: text |
| `canonicalBrand` | `"canonicalBrand": "Sumsung"` | The canonical brand name after normalization of the name from the source.   <br /> Field type: text |
| `colors` | `"colors": [ <br />   "White", <br />   "Black" <br /> ]` | A list of colors available for this product.   <br /> Field type: text |
| `count` | `"count": "30"` | The number of units included in the product packaging.  Can include a description of the unit.   <br /> Field type: keyword |
| `dateAdded` | `"dateAdded": "2017-01-08T19:12:13Z"` | The date this product was first added to the product database.   <br /> Field type: date |
| `dateUpdated` | `"dateUpdated": "2017-01-08T19:12:13Z"` | The most recent date this product was updated or seen by our system.   <br /> Field type: date |
| `descriptions` | `"descriptions": [ <br />   { <br />     "dateSeen": "2017-01-08T19:12:13Z", <br />     "sourceURLs": [    "https://www.domain.com/p/123" <br />     ], <br />     "value": "Now you can enjoy cooking <br /> your favorite food." <br />   }, <br />   { <br />     "sourceURLs": [ <br />  "http://www.website.com/4567" <br />     ], <br />     "value": "The Worthington Pro Grade 20 lb.  <br /> Empty Propane Tank goes with your gas barbecue grill  <br /> (sold separately) and accommodates 4.7 gallons of gas." <br />   } <br /> ]` | A list of descriptions for this product from various sources.
