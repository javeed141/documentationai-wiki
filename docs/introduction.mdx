## What is Datafiniti?

The Datafiniti API gives you access to large, structured datasets across Property, People, Product, and Business data. It provides flexible endpoints for both searching specific data points and enriching your existing datasets.

## Getting Started

To start using the Datafiniti API:

1. **Create an account** at [portal.datafiniti.co](https://portal.datafiniti.co)

2. **Get your API token** from your account settings

3. **Choose a data product** that fits your needs

4. **Make your first API call** using our guides

<Callout kind="tip" collapsed="false">
  New to Datafiniti? Start with our Property Data product to explore how our APIs work.
</Callout>

## Need Help?

If you have questions or need assistance, contact our support team or explore the documentation for your specific data product.
