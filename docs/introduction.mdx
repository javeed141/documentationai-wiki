## What is Datafiniti?

The Datafiniti API provides developers with access to a vast repository of structured data, including Property Data, People Data, Product Data, and Business Data. Our API offers a range of endpoints to suit your needs from searching specifics data point to enrichment of your existing dataset.

## Getting Started

To start using the Datafiniti API:

1. **Create an account** at [portal.datafiniti.co](https://portal.datafiniti.co)

2. **Get your API token** from your account settings

3. **Choose a data product** that fits your needs

4. **Make your first API call** using our guides

<Callout kind="tip" collapsed="false">
  New to Datafiniti? Start with our Property Data product to explore how our APIs work.
</Callout>
