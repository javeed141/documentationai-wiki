Welcome to the Datafiniti API documentation. This guide will help you get started with our data APIs and make the most of our comprehensive datasets.

## What is Datafiniti?

Datafiniti provides access to large-scale, structured datasets through easy-to-use APIs. Our data is collected from millions of sources across the web and normalized into consistent, queryable formats.

## Getting Started

To start using the Datafiniti API:

1. **Create an account** at [portal.datafiniti.co](https://portal.datafiniti.co)
2. **Get your API token** from your account settings
3. **Choose a data product** that fits your needs
4. **Make your first API call** using our guides

<Callout kind="tip">
  New to Datafiniti? Start with our Property Data product to explore how our APIs work.
</Callout>
