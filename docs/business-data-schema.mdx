---
title: Business Data Schema
---

The business data schema below lists all available fields in a business record.  Note that actual business records (actually, most of them) may not have every field present.

| Field Name | Example Value (in JSON) | Description |
| --- | --- | --- |
| `address` | `"address": "3382 30th St"` | The physical street address for this business location.   <br /> Field type: text |
| `categories` | `"categories": [ <br />     "Seafood Restaurant", <br />     "Fish & Seafood Retail", <br />     "restaurant" <br />     "thing" <br /> ]` | A list of category keywords used for this business across multiple sources.   <br /> Field type: text |
| `city` | `"city": "San Diego"` | The city of this business location.   <br /> Field type: text |
| `claimed` | `"claimed": [ <br />   "http://yellowpages.com/", <br />   "http://foursquare.com/" <br /> ]` | A list of websites where the business location has been claimed by the business' owner.   <br /> Field type: keyword |
| `country` | `"country": "US"` | The two-letter country code for the business location's country.   <br /> Field type: text |
| `cuisines` | `"cuisines": [ <br />   "Seafood Restaurants", <br />   "Sandwich Shop", <br />   "Seafood, Vegetarian / Vegan, Mediterranean" <br /> ]` | A list of cuisines offered by this business location.  This field will only show for restaurants or other eating locations.   <br /> Field type: text |
| `dateAdded` | `"dateAdded": "2016-03-24T09:27:02Z"` | The date this business was first added to the business database.   <br /> Field type: date |
| `dateUpdated` | `"dateUpdated": "2017-03-20T20:45:49Z"` | The most recent date this business was updated or seen by our system.   <br /> Field type: date |
| `descriptions` | `"descriptions": [ <br />   { <br />     "dateSeen": "2016-06-18T01:17:41Z", <br />     "sourceURLs": [ <br /> "https://www.zagat.com/r/petes-seafood-and-sandwich-san-diego" <br />     ], <br /> "value": "Sizable made-to-order sandwiches, including lobster rolls, meatball Parmesan heroes and a variety of vegetarian options, are served at this East Coast–style sandwich shop nestled in the heart of North Park." <br />   } <br /> ]` | A list of descriptions for this business from various sources.  Each description contains:   <br /> Field type: nested   <br />    <br /> `dateSeen`: The date when this description has been seen for this business.   <br />   Field type: date   <br />    <br /> `sourceURLs`: A list of source URLs where this specific description was seen.   <br />   Field type: keyword   <br />    <br /> `value`: A unique description for this business.   <br />   Field type: text |
| `domains` | `"domains": [ <br />   "www.superpages.com", <br />   "foursquare.com", <br />   "www.zagat.com" <br /> ]` | A list of each unique domain found in the `sourceURLs` field.   <br /> Field type: domain |
| `emails` | `"emails": [ <br />   "elena@searocketbistro.com" <br /> ]` | A list of emails belonging to the business.   <br /> Field type: text |
| `facebookPageURL` | `"facebookPageURL": "http://www.facebook.com/pages/Sea-Rocket-Bistro/114213985293774"` | A URL pointing to the Facebook page for the business.   <br /> Field type: text |
| `faxes` | `"faxes": [ <br />   "3135619486" <br /> ]` | A list of fax numbers for the business.   <br /> Field type: text |
| `features` | `"features": [ <br />   { <br />     "key": "attire", <br />     "value": [ <br />       "Casual Dress" <br />     ] <br />   }, <br />   { <br />     "key": "parking", <br />     "replace": "true", <br />     "value": [ <br />       "Street Parking", <br />       "Valet" <br />     ] <br />   } <br /> ]` | A list of features associated with this business location.  The feature list can be collected from specific feature sections available on listings for this business.  It can also serve as a catch-all field for miscellaneous characteristics of this business that don't make sense to include as top-level fields.   <br /> Field type: nested   <br />    <br /> Each feature contains:   <br /> `key`: A label for this feature.   <br /> Field type: text   <br />    <br /> `replace`: Flag indicating value field is replaced versus appended.   <br /> Field type: boolean   <br />    <br /> `value`: A list of possible values for this feature.   <br /> Field type: text |
| `geoLocation` | `"geoLocation": "POINT (-119.5094 37.229366)"` | A location value that stores the latitude and longitude in a format that allows geospatial querying.   <br /> Field type: geo_shape   <br />    <br /> The value is stored as Well-Known Text format, which resembles an (x, y) value, i.e. (longitude latitude). |
| `hours` | `"hours": [ <br />   { <br />     "day": "Mon - Fri", <br />     "dept": "Sales", <br />     "hour": "11:00 am - 9:00 pm" <br />   }, <br />   { <br />     "day": "Sat - Sun", <br />     "meal": "Brunch", <br />     "hour": "11:00 am - 10:00 pm" <br />   }, <br /> ]` | The hours of operation for this business location.   <br /> Field type: nested   <br />    <br /> Each hour object may contain:   <br /> `day`: The day(s) for this hour of operation.   <br /> Field type: text   <br />    <br /> `dept`: The department or business unit related to this hour of operation.   <br /> Field type: text   <br />    <br /> `meal`: The mealtime associated with these hours.  Typically reserved for restaurant locations.   <br /> Field type: text   <br /> `hour`: The hours of operation.   <br /> Field type: text |
| `imageURLs` | `"imageURLs": [ <br />   "https://irs0.4sqi.net/img/general/200x200/118688231_EmfdW9_8FpBrxsa5WDVektvkubj-XpCW5ln6s6pnClo.jpg", <br />   "https://irs0.4sqi.net/img/general/200x200/1556291_gjOF8nnudHh_ICSWXoTs75_FxtcKt25nZtwbqjaHtKQ.jpg", <br /> ]` | A list of image URLs associated with this business.   <br /> Field type: text |
| `isClosed` | `"isClosed": "false"` | A `true` or `false` value for whether or not the business location is closed.   <br /> Field type: boolean |
| `keys` | `"keys": [ <br />   "us/ca/sandiego/338230thst/2075732923" <br /> ]` | A list of internal Datafiniti identifiers for this business. | `languagesSpoken` | `"languagesSpoken: [ <br />   "English", <br />   "Spanish" <br /> ]` | A list of the languages spoken at this business.   <br /> Field type: text |
| `latitude` | `"latitude": "32.740654"` | The latitude coordinates for this business location.   <br /> Field type: text |
| `longitude` | `"longitude": "-117.129482"` | The longitude coordinates for this business location.   <br /> Field type: text |
| `menuPageURL` | `"menuPageURL": "http://www.citysearch.com/profile/menu/648254644"` | A URL pointing to the menu for the business.   This field will only show for restaurants or other eating locations.   <br /> Field type: keyword |
| `menus` | `"menus": [ <br />   { <br />     "amountMin": 12, <br />     "amountMax": 12, <br />     "category": "sample menu", <br />     "currency": "USD", <br />     "dateSeen": "2017-03-22T01:03:56Z", <br />     "name": "Fish Carpaccio", <br />     "sourceURLs": [ <br />   "http://www.superpages.com/bp/san-diego-ca/petes-seafood-sandwich-L2615943735.htm" <br />     ] <br />   }, <br />   { <br />     "amountMin": 7, <br />     "amountMax": 7, <br />     "category": "sample menu", <br />     "currency": "USD", <br />     "dateSeen": [ <br />       "2017-03-22T01:03:56Z" <br />     ], <br />     "description": "A dense bread pudding recipe served cold with butterscotch sauce", <br />     "name": "Seasonal Pumpkin Bread Pudding", <br />     "sourceURLs": [ <br />   "http://www.superpages.com/bp/san-diego-ca/petes-seafood-sandwich-L2615943735.htm" <br />     ] <br />   } <br /> ]` | A list of menu items offered by the business.  This field will only show for restaurants or other eating locations.   <br /> Field type: nested   <br />    <br /> Each menu item may include:   <br /> `amountMin`: The minimum price value of the item.   <br /> Field type: float   <br /> `amountMax`: The maximum price value of the item.  This is typically the same as amountMin, except in cases where a price range is listed.   <br /> Field type: float   <br /> `category`: The menu section this item belongs to.   <br /> Field type: text   <br /> `currency`: The currency listed for amountMin and amountMax.   <br /> Field type: keyword   <br /> `dateSeen`: The date when this item was seen.   <br /> Field type: date   <br /> `description`: A description for the item.   <br /> Field type: text   <br /> `name`: The item's name on the menu.   <br /> Field type: text   <br /> `sourceURLs`: A list of URLs where this item was seen.   <br /> Field type: keyword |
| `name` | `"name": "Pete's Seafood & Sandwich"` | The business location's name.   <br /> Field type: text |
| `neighborhoods` | `"neighborhoods": [ <br />   "North Park, Western San Diego", <br />   "North Park" <br /> ]` | A list of neighborhoods where this business is located.  Sometimes an area can belong to multiple neighborhoods, which is why this field is a list.   <br /> Field type: text |
| `numEmployeesMin` | `"numEmployeesMin": "5"` | The lower bound on the number of people possibly employed by this business.   <br /> Field type: integer |
| `numEmployeesMax` | `"numEmployeesMax": "10"` | The upper bound on the number of people employed by this business.   <br /> Field type: integer |
| `numRoom` | `"numRoom": 10` | The number of rooms available at this business.  This field typically shows for hotels or other similar businesses.   <br /> Field type: integer |
| `paymentTypes` | `"paymentTypes":[ <br />   "American Express", <br />   "Mastercard", <br />   "Visa" <br /> ]` | A list of payment options accepted by the business.   <br /> Field type: text |
| `people` | `"people": [ <br />   { <br />     "dateSeen": "2014-08-22T03:48:23Z", <br />     "name": "Cody Jones", <br />     "title": "Owner" <br />   }, <br />   { <br />     "dateSeen": [ <br />       "2014-08-22T03:48:23Z" <br />     ], <br />     "name": "Chaseee Ewing", <br />     "title": "Head Chef" <br />   } <br /> ]` | A list of known people associated with or employed by this business.   <br /> Field type: nested   <br />    <br /> Each person may contain:   <br /> `dateSeen`: The date when this person was last seen.   <br /> Field type: date   <br />    <br /> `email`: The person's email.   <br /> Field type: text   <br />    <br /> `name`: The person's name.   <br /> Field type: text   <br />    <br /> `title`: The person's title.   <br /> Field type: text |
| `phones` | `"phones": [ <br />   "6192558940", <br />   "6195734102" <br /> ]` | A list of phone numbers for the business.   <br /> Field type: keyword |
| `postalCode` | `"postalCode": "92014"` | The postal or zip code of the business' location.   <br /> Field type: keyword |
| `priceRangeCurrency` | `"priceRangeCurrency": "USD"` | The currency associated with the price range fields.   <br /> Field type: keyword |
| `priceRangeMin` | `"priceRangeMin": 25` | The lower bound of the business' price range.  This field is typically used for retail businesses.   <br /> Field type: float |
| `priceRangeMax` | `"priceRangeMax": 40` | The upper bound of the business' price range.  This field is typically used for retail businesses.   <br /> Field type: float |
| `primaryCategories` | `"primaryCategories": [ <br />   "Retail" <br /> ]` | A list of standardized categories to which this business belongs.  Category values can be found [here](/docs/possible-values-for-business-fields).   <br /> Field type: text |
| `productsOrServices` | `"productsOrServices": [ <br />   "Food", <br />   "Catering" <br /> ]` | A list of products or services offered by this business.   <br /> Field type: text |
| `province` | `"province": "CA"` | The province or state for this business location.   <br /> Field type: text |
| `revenueCurrency` | `"revenueCurrency": "USD"` | The currency associated with the revenue fields.   <br /> Field type: keyword |
| `revenueMin` | `"revenueMin": 30000` | The minimum known value of the business' annual revenue.   <br /> Field type: integer |
| `revenueMax` | `"revenueMax": 100000` | The maximum known value of the business' annual revenue.   <br /> Field type: integer |

Chunyi Hsu"   <br />   }   <br /> ]\` | A list of reviews for this business location.   <br /> Field type: nested   <br />    <br /> Review objects may contain:   <br /> `date`: The date the review was posted.   <br /> Field type: date   <br />    <br /> `dateSeen`: The date when the review was seen by Datafiniti.   <br /> Field type: date   <br />    <br /> `doRecommend`: A true/false for whether or not the reviewer recommends the product.   <br /> Field type: boolean   <br />    <br /> `id`: The website ID associated with this review.   <br /> Field type: keyword   <br />    <br /> `rating`: A 1 to 5 star value for the review.   <br /> Field type: double   <br />    <br /> `sourceURLs`: A list of URLs where this review was seen.   <br /> Field type: keyword   <br />    <br /> `text`: The full (or available) text of the review.   <br /> Field type: text   <br />    <br /> `title`: The review's title.   <br /> Field type: text   <br />    <br /> `userCity`: The reviewer's city.   <br /> Field type: text   <br />    <br /> `username`: The reviewer's username.   <br /> Field type: text   <br />    <br /> `userProvince`: The reviewer's province or state.   <br /> Field type: text |
