---
title: "Count"
description: "Get record counts without downloading data from the Datafiniti Property Data API."
---

The count feature allows you to see how many records match your query without downloading any data.

## Getting a count

Set `num_records` to `0` to get only the count:

```json
{
  "query": "province:TX AND mostRecentStatus:\"For Sale\"",
  "num_records": 0
}
```

## Response

```json
{
  "num_found": 125432,
  "total_cost": 0,
  "records": []
}
```

| Field | Description |
|-------|-------------|
| `num_found` | Total records matching your query |
| `total_cost` | Credits used (0 for count-only requests) |
| `records` | Empty array |

## Use cases

### Estimate dataset size

Before downloading, check how many records you'll receive:

```json
{
  "query": "province:CA AND propertyType:\"Single Family Dwelling\" AND mostRecentStatus:\"For Sale\"",
  "num_records": 0
}
```

### Validate query syntax

Test that your query returns expected results:

```json
{
  "query": "city:Austin AND numBedroom:[3 TO 5] AND prices.amountMin:[200000 TO 500000]",
  "num_records": 0
}
```

Track how inventory changes over time by running count queries periodically.
