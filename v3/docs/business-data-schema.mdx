---
title: Business Data Schema
---

The business data schema below lists all available fields in a business record.  Note that actual business records (actually, most of them) may not have every field present.

| Field Name | Example Value (in JSON) | Description |
| --- | --- | --- |
| `address` | ``` <br/> "address": "3382 30th St" <br/> ``` | The physical street address for this business location. |
| `careerPageURL` | ``` <br/> "careerPageURL": "http://www.searocketbistro.com/employment" <br/> ``` | A URL pointing to the career page for the business. |
| `categories` | ``` <br/> "categories": [ <br/>   "Seafood Restaurant", <br/>   "Fish & Seafood Retail", <br/>   "restaurant" <br/> ] <br/> ``` | A list of category keywords used for this business across multiple sources. |
| `city` | ``` <br/> "city": "San Diego" <br/> ``` | The city of this business location. |
| `claimed` | ``` <br/> "claimed": [ <br/>   "http://yellowpages.com/", <br/>   "http://foursquare.com/" <br/> ] <br/> ``` | A list of websites where the business location has been claimed by the business' owner. |
| `country` | ``` <br/> "country": "US" <br/> ``` | The two-letter country code for the business location's country. |
| `cuisines` | ``` <br/> "cuisines": [ <br/>   "Seafood Restaurants", <br/>   "Sandwich Shop", <br/>   "Seafood, Vegetarian / Vegan, Mediterranean" <br/> ] <br/> ``` | A list of cuisines offered by this business location.  This field will only show for restaurants or other eating locations. |
| `dateAdded` | ``` <br/> "dateAdded": "2016-03-24T09:27:02Z" <br/> ``` | The date this business was first added to the business database. |
| `dateUpdated` | ``` <br/> "dateUpdated": "2017-03-20T20:45:49Z" <br/> ``` | The most recent date this business was updated or seen by our system. |
| `descriptions` | ``` <br/> "descriptions": [ <br/>   { <br/>     "dateSeen": [ <br/>       "2016-06-18T01:17:41Z" <br/>     ], <br/>     "sourceURLs": [ <br/>       "https://www.zagat.com/r/petes-seafood-and-sandwich-san-diego" <br/>     ], <br/>     "value": "Sizable made-to-order sandwiches, including lobster rolls, meatball Parmesan heroes and a variety of vegetarian options, are served at this East Coast–style sandwich shop nestled in the heart of North Park." <br/>   } <br/> ] <br/> ``` | A list of descriptions for this business from various sources.  Each description contains: <br/>  <br/> * `dateSeen`: A list of dates when this description has been seen for this business. <br/> * `sourceURLs`: A list of source URLs where this specific description was seen. <br/> * `value`: A unique description for this business. |
| `emails` | ``` <br/> "emails": [ <br/>   "elena@searocketbistro.com" <br/> ] <br/> ``` | A list of emails belonging to the business. |
| `facebookPageURL` | ``` <br/> "facebookPageURL": "http://www.facebook.com/pages/Sea-Rocket-Bistro/114213985293774" <br/> ``` | A URL pointing to the Facebook page for the business. |
| `faxes` | ``` <br/> "faxes": [ <br/>   "3135619486" <br/> ] <br/> ``` | A list of fax numbers for the business. |
| `features` | ``` <br/> "features": [ <br/>   { <br/>     "key": "attire", <br/>     "value": [ <br/>       "Casual Dress" <br/>     ] <br/>   }, <br/>   { <br/>     "key": "parking", <br/>     "value": [ <br/>       "Street Parking", <br/>       "Valet" <br/>     ] <br/>   } <br/> ] <br/> ``` | A list of features associated with this business location.  The feature list can be collected from specific feature sections available on listings for this business.  It can also serve as a catch-all field for miscellaneous characteristics of this business that don't make sense to include as top-level fields. <br/>  <br/> Each feature contains: <br/> * `key`: A label for this feature. <br/> * `value`: A list of possible values for this feature. |
| `hours` | ``` <br/> "hours": [ <br/>   { <br/>     "day": "Mon - Fri", <br/>     "hour": "11:00 am - 9:00 pm" <br/>   }, <br/>   { <br/>     "day": "Sat - Sun", <br/>     "hour": "11:00 am - 10:00 pm" <br/>   }, <br/> ] <br/> ``` | The hours of operation for this business location. <br/>  <br/> Each hour object may contain: <br/> * `day`: The day(s) for this hour of operation. <br/> * `dept`: The department or business unit related to this hour of operation. <br/> * `hour`: The hours of operation. |
| `imageURLs` | ``` <br/> "imageURLs": [ <br/>   "https://irs0.4sqi.net/img/general/200x200/118688231_EmfdW9_8FpBrxsa5WDVektvkubj-XpCW5ln6s6pnClo.jpg", <br/>   "https://irs0.4sqi.net/img/general/200x200/1556291_gjOF8nnudHh_ICSWXoTs75_FxtcKt25nZtwbqjaHtKQ.jpg", <br/> ] <br/> ``` | A list of image URLs associated with this business. |
| `isClosed` | ``` <br/> "isClosed": "false" <br/> ``` | A `true` or `false` value for whether or not the business location is closed. |
| `keys` | ``` <br/> "keys": [ <br/>   "us/ca/sandiego/338230thst/2075732923" <br/> ] <br/> ``` | A list of internal Datafiniti identifiers for this business.  The `keys` field is used to merge raw data from individual sources into the master Datafiniti record.  [Learn more about how this works](https://datafiniti.co). |
| `languagesSpoken` | ``` <br/> "languagesSpoken: [ <br/>   "English", <br/>   "Spanish" <br/> ] <br/> ``` | A list of the languages spoken at this business. |
| `latitude` | ``` <br/> "latitude": "32.740654" <br/> ``` | The latitude coordinates for this business location. |
| `longitude` | ``` <br/> "longitude": "-117.129482" <br/> ``` | The longitude coordinates for this business location. |
| `menuPageURL` | ``` <br/> "menuPageURL": "http://www.citysearch.com/profile/menu/648254644" <br/> ``` | A URL pointing to the menu for the business.   This field will only show for restaurants or other eating locations. |
| `menus` | ``` <br/> "menus": [ <br/>   { <br/>     "amountMin": 12, <br/>     "amountMax": 12, <br/>     "category": "sample menu", <br/>     "currency": "USD", <br/>     "dateSeen": [ <br/>       "2017-03-22T01:03:56Z" <br/>     ], <br/>     "name": "Fish Carpaccio", <br/>     "sourceURLs": [ <br/>   "http://www.superpages.com/bp/san-diego-ca/petes-seafood-sandwich-L2615943735.htm" <br/>     ] <br/>   }, <br/>   { <br/>     "amountMin": 7, <br/>     "amountMax": 7, <br/>     "category": "sample menu", <br/>     "currency": "USD", <br/>     "dateSeen": [ <br/>       "2017-03-22T01:03:56Z" <br/>     ], <br/>     "description": "A dense bread pudding recipe served cold with butterscotch sauce", <br/>     "name": "Seasonal Pumpkin Bread Pudding", <br/>     "sourceURLs": [ <br/>   "http://www.superpages.com/bp/san-diego-ca/petes-seafood-sandwich-L2615943735.htm" <br/>     ] <br/>   } <br/> ] <br/> ``` | A list of menu items offered by the business.  This field will only show for restaurants or other eating locations. <br/>  <br/> Each menu item may include: <br/> * `amountMin`: The minimum price value of the item. <br/> * `amountMax`: The maximum price value of the item.  This is typically the same as amountMin, except in cases where a price range is listed. <br/> * `category`: The menu section this item belongs to. <br/> * `currency`: The currency listed for amountMin and amountMax. <br/> * `dateSeen`: A list of dates when this item was seen. <br/> * `description`: A description for the item. <br/> * `name`: The item's name on the menu. <br/> * `sourceURLs`: A list of URLs where this item was seen. |
| `name` | ``` <br/> "name": "Pete's Seafood & Sandwich" <br/> ``` | The business location's name. |
| `neighborhoods` | ``` <br/> "neighborhoods": [ <br/>   "North Park, Western San Diego", <br/>   "North Park" <br/> ] <br/> ``` | A list of neighborhoods where this business is located.  Sometimes an area can belong to multiple neighborhoods, which is why this field is a list. |
| `numEmployeesMin` | ``` <br/> "numEmployeesMin": "5" <br/> ``` | The lower bound on the number of people possibly employed by this business. |
| `numEmployeesMax` | ``` <br/> "numEmployeesMax": "10" <br/> ``` | The upper bound on the number of people employed by this business. |
| `numRoom` | ``` <br/> "numRoom": 10 <br/> ``` | The number of rooms available at this business.  This field typically shows for hotels or other similar businesses. |
| `paymentTypes` | ``` <br/> "paymentTypes":[ <br/>   "American Express", <br/>   "Mastercard", <br/>   "Visa" <br/> ] <br/> ``` | A list of payment options accepted by the business. |
| `people` | ``` <br/> "people": [ <br/>   { <br/>     "dateSeen": [ <br/>       "2014-08-22T03:48:23Z" <br/>     ], <br/>     "name": "Cody Jones", <br/>     "title": "Owner" <br/>   }, <br/>   { <br/>     "dateSeen": [ <br/>       "2014-08-22T03:48:23Z" <br/>     ], <br/>     "name": "Chaseee Ewing", <br/>     "title": "Head Chef" <br/>   } <br/> ] <br/> ``` | A list of known people associated with or employed by this business. <br/>  <br/> Each person may contain: <br/> * `dateSeen`: A list of dates when this person was seen. <br/> * `name`: The person's name. <br/> * `title`: The person's title. |
| `phones` | ``` <br/> "phones": [ <br/>   "6192558940", <br/>   "6195734102" <br/> ] <br/> ``` | A list of phone numbers for the business. |
| `postalCode` | ``` <br/> "postalCode": "92014" <br/> ``` | The postal or zip code of the business' location. |
| `priceRangeCurrency` | ``` <br/> "priceRangeCurrency": "USD" <br/> ``` | The currency associated with the price range fields. |
| `priceRangeMin` | ``` <br/> "priceRangeMin": 25 <br/> ``` | The lower bound of the business' price range.  This field is typically used for retail businesses. |
| `priceRangeMax` | ``` <br/> "priceRangeMax": 40 <br/> ``` | The upper bound of the business' price range.  This field is typically used for retail businesses. |
| `productsOrServices` | ``` <br/> "productsOrServices": [ <br/>   "Food", <br/>   "Catering" <br/> ] <br/> ``` | A list of products or services offered by this business. |
| `province` | ``` <br/> "province": "CA" <br/> ``` | The province or state for this business location. |
| `revenueCurrency` | ``` <br/> "revenueCurrency": "USD" <br/> ``` | The currency associated with the revenue fields. |
| `revenueMin` | ``` <br/> "revenueMin": 30000 <br/> ``` | The minimum known value of the business' annual revenue. |
| `revenueMax` | ``` <br/> "revenueMax": 100000 <br/> ``` | The maximum known value of the business' annual revenue. |
| `reviews` | ``` <br/> "reviews": [ <br/>   { <br/>     "date": "2016-02-27T00:00:00Z", <br/>     "dateAdded": "2017-03-20T20:45:49Z", <br/>     "dateSeen": [ <br/>       "2016-05-17T00:00:00Z", <br/>       "2016-05-13T00:00:00Z", <br/>     ], <br/>     "sourceURLs": [ <br/>     "https://foursquare.com/v/petes-seafood--sandwich/56391bfacd105fac6c403c63" <br/>     ], <br/>     "text": "best fried ipswich clams in san diego!  large portions- not just the 6-8 clams you get at other local eateries. great service! i will be back!", <br/>     "username": "Colleen Alarcon" <br/>   }, <br/>   { <br/>     "date": "2016-02-15T00:00:00Z", <br/>     "dateAdded": "2017-03-20T20:45:49Z", <br/>     "dateSeen": [ <br/>       "2016-05-17T00:00:00Z", <br/>       "2016-05-13T00:00:00Z", <br/>     ], <br/>     "sourceURLs": [ <br/>    <br/>    <br/>   "https://foursquare.com/v/petes-seafood--sandwich/56391bfacd105fac6c403c63" <br/>     ], <br/>     "text": "the lobster roll is simply amazing!!!", <br/>     "username": "P. Chunyi Hsu" <br/>   } <br/> ] <br/> ``` | A list of reviews for this business location. <br/>  <br/> Review objects may contain: <br/> * `date`: The date the review was posted. <br/> * `dateAdded`: The date the review was added to Datafiniti.  <br/> * `dateSeen`: A list of dates when the review was seen by Datafiniti. <br/> * `doRecommend`: A true/false for whether or not the reviewer recommends the product. <br/> * `id`: The website ID associated with this review. <br/> * `rating`: A 1 to 5 star value for the review. <br/> * `sourceURLs`: A list of URLs where this review was seen. <br/> * `text`: The full (or available) text of the review. <br/> * `title`: The review's title. <br/> * `userCity`: The reviewer's city. <br/> * `username`: The reviewer's username. <br/> * `userProvince`: The reviewer's province or state. |
| `rooms` | ``` <br/> "rooms": [ <br/>   { <br/>     "amountMin": 90, <br/>     "amountMax": 90, <br/>     "capacity": 3, <br/>     "currency": "USD", <br/>     "dateSeen": [ <br/>       "2016-05-04T00:00:00Z" <br/>     ], <br/>     "roomType": "Standard Room" <br/>   }, <br/>   { <br/>     "amountMin": 119, <br/>     "amountMax": 119, <br/>     "capacity": 4, <br/>     "currency": "USD", <br/>     "dateSeen": [ <br/>       "2016-05-04T00:00:00Z" <br/>     ], <br/>     "roomType": "Superior Room" <br/>   } <br/> ] <br/> ``` | A list of room types available at the business' location.  This field is typically associated with hotels and other similar businesses. <br/>  <br/> Room objects may contain: <br/> * `amountMin`: The minimum price value listed for the room. <br/> * `amountMax`: The maximum price value listed. <br/> * `capacity`: The maximum number of people that can occupy this room. <br/> * `currency`: The currency listed for amountMin and amountMax. <br/> * `dateSeen`: A list of dates when this room was seen. <br/> * `roomType`: A classification for the type of room. |
| `sic` | ``` <br/> "sic": "5610" <br/> ``` | The SIC code for the business.  [Learn more about SIC codes](https://en.wikipedia.org/wiki/Standard_Industrial_Classification). |
| `sourceURLs` | ``` <br/> "sourceURLs": [ <br/>   "http://www.superpages.com/bp/san-diego-ca/petes-seafood-sandwich-L2615943735.htm", <br/>   "https://foursquare.com/v/petes-seafood--sandwich/56391bfacd105fac6c403c63", <br/>   "https://www.zagat.com/r/petes-seafood-and-sandwich-san-diego", <br/> ] <br/> ``` | A list of URLs used to generate data for this business location. |
| `twitter` | ``` <br/> "twitter": "sdpark2park" <br/> ``` | The twitter username for the business. |
| `websites` | ``` <br/> "websites": [ <br/>   "http://www.bostonpetes.com/", <br/>   "http://www.searocketbistro.com" <br/> ] <br/> ``` | A list of websites for the business. |
| `yearIncorporated` | ``` <br/> "yearIncorporated": 2004  <br/> ``` | The year the business was incorporated. |
| `yearOpened` | ``` <br/> "yearOpened": 2005 <br/> ``` | The year the business opened. |
