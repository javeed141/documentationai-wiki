---
title: Product Data Schema
---

The product data schema below lists all available fields in a product record.  Note that actual product records (actually, most of them) may not have every field present.

| Field Name | Example Value (in JSON) | Description |
| --- | --- | --- |
| `asins` | ``` <br/> "asins": [ <br/>   "B0009XCFRE" <br/> ] <br/> ``` | A list of ASINs (Amazon identifiers) used for this product. |
| `brand` | ``` <br/> "brand": "Worthington" <br/> ``` | The brand name of this product. |
| `categories` | ``` <br/> "categories": [ <br/>   "Home Improvement", <br/>   "Heating, Cooling, & Air Quality", <br/>   "Heaters", <br/>   "Propane Heaters" <br/> ] <br/> ``` | A list of category keywords used for this product across multiple sources. |
| `colors` | ``` <br/> "colors": [ <br/>   "White", <br/>   "Black" <br/> ] <br/> ``` | A list of colors available for this product. |
| `count` | ``` <br/> "count": 1 <br/> ``` | The number of units included in the product packaging. |
| `dateAdded` | ``` <br/> "dateAdded": "2017-01-08T19:12:13Z" <br/> ``` | The date this product was first added to the product database. |
| `dateUpdated` | ``` <br/> "dateUpdated": "2017-01-08T19:12:13Z" <br/> ``` | The most recent date this product was updated or seen by our system. |
| `descriptions` | ``` <br/> "descriptions": [ <br/>   { <br/>     "dateSeen": [ <br/>       "2017-01-08T19:12:13Z" <br/>     ] <br/>     "sourceURLs": [  <br/>       "https://www.domain.com/p/123" <br/>     ], <br/>     "value": "Now you can enjoy cooking your favorite food." <br/>   }, <br/>   { <br/>     "sourceURLs": [ <br/>       "http://www.website.com/4567" <br/>     ], <br/>     "value": "The Worthington Pro Grade 20 lb. Empty Propane Tank goes with your gas barbecue grill (sold separately) and accommodates 4.7 gallons of gas." <br/>   } <br/> ] <br/> ``` | A list of descriptions for this product from various sources.  Each description contains: <br/>  <br/> * `dateSeen`: A list of dates when this description has been seen selling this product. <br/> * `sourceURLs`: A list of source URLs where this specific description was seen. <br/> * `value`: A unique description for this product. |
| `dimension` | ``` <br/> "dimension": "23 in x 43.7 in x 30 in" <br/> ``` | The length, width, and height of this product.  Units included. |
| `ean` | ``` <br/> "ean": [ <br/>   "0014045125963", <br/>   "0014045125965" <br/> ] <br/> ``` | The EAN codes for this product.  There may be more than one EAN in the list when a product has multiple variations, <br/>  each using a different EAN. <br/>   [Learn more about EANs](https://en.wikipedia.org/wiki/International_Article_Number). |
| `features` | ``` <br/> "features": [ <br/>   { <br/>     "key": "Count", <br/>     "value": [ <br/>       "1" <br/>     ] <br/>   }, <br/>   { <br/>     "key": "Dimensions", <br/>     "value": [ <br/>       "12.5 x 12.5 x 18.5 in" <br/>     ] <br/>   } <br/> ] <br/> ``` | A list of features associated with this product.  The feature list can be collected from specific feature sections available on product listings for the product.  It can also serve as a catch-all field for miscellaneous characteristics of this product that don't make sense to include as top-level fields. <br/>  <br/> Each feature contains: <br/> * `key`: A label for this feature. <br/> * `value`: A list of possible values for this feature. |
| `flavors` | ``` <br/> "flavors": [ <br/>   "Fruit Punch", <br/>   "Berry", <br/>   "Grape" <br/> ] <br/> ``` | A list of flavors available for this product. |
| `imageURLs` | ``` <br/> "imageURLs": [ <br/>   "https://i5.images.com/asr.jpeg" <br/> ] <br/> ``` | A list of image URLs for this product. |
| `isbn` | ``` <br/> "isbn": "3882215542" <br/> ``` | The ISBN code for this product.  [Learn more about ISBNs](https://en.wikipedia.org/wiki/International_Standard_Book_Number). |
| `keys` | ``` <br/> "keys": [ <br/>   "014045125963", <br/>   "0014045125963", <br/>   "worthington/299581" <br/> ] <br/> ``` | A list of internal Datafiniti identifiers for this product.  The `keys` field is used to merge raw data from individual sources into the master Datafiniti record.  [Learn more about how this works](https://datafiniti.co). |
| `manufacturer` | ``` <br/> "manufacturer": "Worthington" <br/> ``` | The manufacturer of this product. |
| `manufacturerNumber` | ``` <br/> "manufacturerNumber": "299581" <br/> ``` | The manufacturer or model number of this product. |
| `merchants` | ``` <br/> "merchants": [ <br/>   { <br/>     "address": "2801 S Ih 35", <br/>     "availability": "...", <br/>     "city": "Austin", <br/>     "country": "US", <br/>     "dateSeen": [ <br/>       "2017-01-08T19:12:13Z" <br/>     ], <br/>     "isPrivateSeller": "false" <br/>     "name": "Zappysales", <br/>     "phone": "5127131234", <br/>     "postalCode": "78701", <br/>     "province": "TX" <br/>   }, <br/>   { <br/>     "dateSeen": [ <br/>       "2017-01-08T19:12:13Z" <br/>     ], <br/>     "name": "Next Day Mro, Llc" <br/>   } <br/>  ] <br/> ``` | A list of merchants selling this product.  These are typically third-party merchants found on e-commerce websites. <br/>  <br/> Merchant objects can contain: <br/> * `address`: The merchant's physical address. <br/> * `availability`: A description of the product's availability with this merchant. <br/> * `city`: The merchant's city. <br/> * `dateSeen`: A list of dates when this merchant has been seen selling this product. <br/> * `isPrivateSeller`: True if the merchant is an individual. False if it's a company. <br/> * `name`: The merchant's name. <br/> * `phone`: The merchant's phone number. <br/> * `postalCode`: The merchant's postal or zip code. <br/> * `province`: The merchant's province or state. |
| `name` | ``` <br/> "name": "Worthington 20-lb Tank" <br/> ``` | The product's name. |
| `prices` | ``` <br/> "prices": [ <br/>   { <br/>     "amountMin": 59.96, <br/>     "amountMax": 59.96, <br/>     "availability": "...", <br/>     "color": "...", <br/>     "condition": "New", <br/>     "count": "...", <br/>     "currency": "USD", <br/>     "dateAdded": "2017-01-08T19:12:13Z", <br/>     "dateSeen": [ <br/>       "2016-06-10T00:00:00Z" <br/>     ], <br/>     "flavor": "...", <br/>     "isSale": "false", <br/>     "isSold": "false", <br/>     "merchant": "domain.com", <br/>     "offer": "22% off", <br/>     "returnPolicy": "90-Day", <br/>     "shipping": "FREE shipping", <br/>     "size": "...", <br/>     "sourceURLs": [ <br/>       "https://www.domain.com/p/123", <br/>      "warranty": "..." <br/>     ] <br/>   }, <br/>   { <br/>     "amountMin": 62.06, <br/>     "amountMax": 72.06, <br/>     "condition": "new", <br/>     "currency": "USD", <br/>     "dateAdded": "2017-01-08T19:12:13Z", <br/>     "dateSeen": [ <br/>       "2016-10-28T00:00:00Z" <br/>     ], <br/>     "merchant": "MRO - website.com", <br/>     "sourceURLs": [ <br/>       "http://www.website.com/4567" <br/>     ] <br/>   } <br/> ] <br/> ``` | A list of prices for this product. <br/>  <br/> Price objects may contain: <br/> * `amountMin`: The minimum price value listed. <br/> * `amountMax`: The maximum price value listed.  This is typically the same as amountMin, except in cases where a price range is listed. <br/> * `availability`: The product's availability at this price. <br/> * `color`: The color associated with this price. <br/> * `condition`: The condition of the product when being sold at this price. <br/> * `currency`: The currency listed for amountMin and amountMax. <br/> * `dateAdded`: The date this price was first added to the prices list. <br/> * `dateSeen`: A list of dates when this price was seen. <br/> * `flavor`: The flavor associated with this price. <br/> * `isSale`: A true/false for whether or not this price is a sale/discounted price. <br/> * `isSold`: A true/flase for whether or not the product has been sold at this price (typically reserved for vehicle listings). <br/> * `merchant`: The merchant and/or website selling at this price. <br/> * `offer`: Any special offer associated with this price. <br/> * `returnPolicy`: The return policy associated with this price. <br/> * `shipping`: The shipping terms associated with this price. <br/> * `sourceURLs`: A list of URLs where this price was seen. <br/> * `warranty`: The warranty associated with this price. |
| `quantities` | ``` <br/> "quantities": [  <br/>   { <br/>     "dateSeen": [ <br/>       "2013-06-02T16:00:00Z", <br/>       "2014-05-01T14:00:00Z" <br/>     ],  <br/>     "sourceURLs": [ <br/>       "http://www.website.com/4567" <br/>     ], <br/>     "value": 20 <br/>   } <br/> ] <br/> ``` | A list of available quantities for this product.  Quantities indicate how many units of the product are available from a specific source at a given time. <br/>  <br/> Quantity objects may contain: <br/> * `dateSeen`: A list of dates when this quantity was seen. <br/> * `sourceURLs`: A list of URLs where this quantity was seen. <br/> * `value`: The actual quantity value. |
| `reviews` | ``` <br/> "reviews": [ <br/>   { <br/>     "date": "2015-06-16T00:00:00Z", <br/>     "dateAdded": "2017-01-08T19:12:13Z", <br/>     "dateSeen": [ <br/>       "2016-05-15T00:00:00Z" <br/>     ], <br/>     "didPurchase": true, <br/>     "doRecommend": true, <br/>     "id": "1234" <br/>     "numHelpful": 4, <br/>     "rating": 4, <br/>     "sourceURLs": [ <br/>         "http://www.website.com/4567" <br/>     ], <br/>     "text": "Nice tank at a below average cost for propane tanks.", <br/>     "title": "Solid Propane Tank", <br/>     "userCity": "...", <br/>     "username": "Cole33", <br/>     "userProvince": "..." <br/>   }, <br/>   { <br/>     "date": "2015-01-06T00:00:00Z", <br/>     "dateAdded": "2017-01-08T19:12:13Z", <br/>     "dateSeen": [ <br/>       "2016-05-15T00:00:00Z" <br/>     ], <br/>     "rating": 5, <br/>     "sourceURLs": [ <br/>         "https://www.domain.com/p/123" <br/>       ], <br/>     "text": "Holds propane and does not leak. Completed the given gift.", <br/>     "title": "tanked", <br/>     "username": "joel01" <br/>   } <br/> ] <br/> ``` | A list of reviews for this product. <br/>  <br/> Review objects may contain: <br/> * `date`: The date the review was posted. <br/> * `dateAdded`: The date the review was added to Datafiniti.  <br/> * `dateSeen`: A list of dates when the review was seen by Datafiniti. <br/> * `didPurchase`: A true/false for whether or not the reviewer did purchase the product. <br/> * `doRecommend`: A true/false for whether or not the reviewer recommends the product. <br/> * `id`: The website ID associated with this review. <br/> * `numHelpful`: The number of people that found this review helpful. <br/> * `rating`: A 1 to 5 start value for the review. <br/> * `sourceURLs`: A list of URLs where this review was seen. <br/> * `text`: The full (or available) text of the review. <br/> * `title`: The review's title. <br/> * `userCity`: The reviewer's city. <br/> * `username`: The reviewer's username. <br/> * `userProvince`: The reviewer's province or state. |
| `sizes` | ``` <br/> "sizes": [ <br/>   "Large", <br/>   "Medium", <br/>   "Small" <br/> ] <br/> ``` | A list of sizes available for this product. |
| `skus` | ``` <br/> "skus": [ <br/>   { <br/>     "sourceURLs": [ <br/>       "http://www.web.com/p/123" <br/>     ], <br/>     "value": "1234XK6" <br/>   } <br/> ] <br/> ``` | A list of SKUs for this product.  SKUs are typically specific to individual retailers or websites. <br/>  <br/> SKU objects may contain: <br/> * `sourceURLs`: A list of URLs where this SKU was seen. <br/> * `value`: The actual SKU. |
| `sourceURLs` | ``` <br/> "sourceURLs": [ <br/>   "https://www.domain.com/p/123", <br/>   "http://www.website.com/4567" <br/> ] <br/> ``` | A list of URLs used to generate data for this product. |
| `upc` | ``` <br/> "upc": [ <br/>   "014045125963", <br/>   "014045125961" <br/> ] <br/> ``` | The UPC code for this product.  <br/>   There may be more than one UPC in the list when a product has multiple variations, <br/>  each using a different UPC.  [Learn more about UPCs](https://en.wikipedia.org/wiki/Universal_Product_Code). |
| `vin` | ``` <br/> "vin": "1FTMF1ET8EFB00361" <br/> ``` | The VIN code for this product.  This is specific to car listings.  [Learn more about VINs](https://en.wikipedia.org/wiki/Vehicle_identification_number). |
| `websiteIDs` | ``` <br/> "websiteIDs": [ <br/>   "domain.com-123", <br/>   "website.com-4567" <br/> ] <br/> ``` | A list of website IDs for this product.  Website IDs are tied to a specific online retailer.  They are the unique identifier used by website.  They are used by Datafiniti when other unique identifiers are not available for a product. |
| `weight` | ``` <br/> "weight": "20 lbs" <br/> ``` | The weight of the product.  Units included. |
