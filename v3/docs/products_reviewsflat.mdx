---
title: products_reviewsFlat
---

The `products_reviewsFlat` view should be used when you want to look at individual reviews for each product in your data set in a CSV format.  It will split each review for a product into its own row.  All other fields will repeat in each row.


## Included Fields

| Field | Included Sub-fields |
| --- | --- |
| `asins` | N/A |
| `brand` | N/A |
| `categories` | N/A |
| `colors` | N/A |
| `count` | N/A |
| `dateAdded` | N/A |
| `dateUpdated` | N/A |
| `descriptions` | `dateSeen` <br/> `sourceURLs` <br/> `value` |
| `dimension` | N/A |
| `ean` | N/A |
| `features` | `key` <br/> `value` |
| `flavors` | N/A |
| `imageURLs` | N/A |
| `isbn` | N/A |
| `keys` | N/A |
| `manufacturer` | N/A |
| `manufacturerNumber` | N/A |
| `merchants` | `address` <br/> `availability` <br/> `city` <br/> `country` <br/> `dateSeen` <br/> `name` <br/> `phone` <br/> `postalCode` <br/> `province` |
| `name` | N/A |
| `prices` | `amountMin` <br/> `amountMax` <br/> `availability` <br/> `color` <br/> `condition` <br/> `count` <br/> `currency` <br/> `dateAdded` <br/> `dateSeen` <br/> `flavor` <br/> `isSale` <br/> `merchant` <br/> `offer` <br/> `returnPolicy` <br/> `shipping` <br/> `size` <br/> `source` <br/> `sourceURLs` <br/> `warranty` |
| `reviews` | `date` <br/> `dateAdded` <br/> `dateSeen` <br/> `didPurchase` <br/> `doRecommend` <br/> `numHelpful` <br/> `rating` <br/> `sourceURLs` <br/> `text` <br/> `title` <br/> `userCity` <br/> `username` <br/> `userProvince` |
| `sizes` | N/A |
| `skus` | `sourceURLs` <br/> `value` |
| `sourceURLs` | N/A |
| `upc` | N/A |
| `vin` | N/A |
| `websiteID` | N/A |
| `weight` | N/A |
