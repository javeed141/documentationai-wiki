---
title: Available Views for Business Data
---

The `view` parameter in the API determines which fields are returned by your data request and the order in which they're shown.  It also determines if multi-valued fields will flatten or expand their individual elements into multiple entries.

Note that each `view` is available in either `json` or `csv` format.  The `format` parameter you use will determine which one is used.

Available views for business data:

* [businesses_all](/v3/docs/businesses_all)
* [businesses_all_menusFlat](/v3/docs/businesses_all_menusflat)
* [businesses_all_nested](/v3/docs/businesses_all_nested)
* [businesses_all_nested_no_reviews](/v3/docs/businesses_all_nested_no_reviews)
* [businesses_all_reviewsFlat](/v3/docs/businesses_all_reviewsflat)
* [businesses_basic](/v3/docs/businesses_basic)
