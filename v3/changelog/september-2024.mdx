## New Realtor License Types

You can now query and identify realtor information based on state realty licenses. The following license types have been added:

- Delaware (DERED), Illinois (IDFPR), Vermont (VTREC), Wyoming (WYREC)
- Virginia (VADPOR), New Jersey (NJDOBI), Hawaii (HDCCA)
- Oregon (OREA), Maryland (MDMREC), Wisconsin (WIDSPS)

## Portal Improvements

- **Keyword Search** — Search property data via feature key, feature value, or descriptions
- **Sequential Downloads** — "Download All" now downloads files one at a time for a cleaner browsing experience
