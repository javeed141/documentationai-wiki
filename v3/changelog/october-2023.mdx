## New Property Statuses

- **Auction** — Properties scheduled for auction
- **Pre-Foreclosed** — Properties in pre-foreclosure status
- **Foreclosed** — Foreclosed/distressed properties

## New Top-Level Property Fields

- `congressionalDistrictHouse` — Congressional district of the property
- `legalRange` — Legal range of the property
- `currentOwnerType` — Current owner type (e.g., "Company")
- `instrumentNumber` — Instrument number of the property
