## Autotrace — Combined People & Property Data

You can now combine People Data and Property Data in a single API call using the new `auto_trace` parameter. This enhancement to the standard property search endpoint pulls property records with homeowner contact information in one request.

Learn more in the [Autotrace documentation](/v3/docs/autotrace).

## New County Assessor Data

Added county assessor data for the following counties:

- **Florida** — Lee County
- **Georgia** — Dekalb County
- **New Jersey** — Hudson County, Bergen County, Middlesex County

## Improved APN / Parcel Number Data

Updated how APN/Parcel numbers are imported. You can now query the `parcelNumbers` field with greater accuracy, including a new `year` subfield for date-based filtering.

## People Data Portal Redesign

Updated People Data portal design with auto-search names, easy-to-use filters, and one-click common searches at [portal.datafiniti.co/people-data](https://portal.datafiniti.co/people-data).
