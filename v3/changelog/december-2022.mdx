## Property Source Updates

Added CBRE.ca and CBRE.com for commercial properties — over 20,000 new records.

## Business Source Updates

- Added allmenus.com for delivery and fast service restaurants
- Added Amazon.co.uk and Amazon.ca

## New Product Field: `canonicalBrand`

New `canonicalBrand` top-level field provides normalized brand names. The `brand` field now contains the raw scraped text.

## Enrichment API Improvements

Improved enrichment API across all data types with 4.5x speed increase for starting and finishing enrichment calls.

## Fixes

- Cleaned ~100,000 improperly normalized property records
- Fixed zoopla.co.uk crawler HTML parsing issue
