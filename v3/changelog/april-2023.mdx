## New Delisted Status Types

Eight new off-market status types added:

- Delisted (Withdrawn), Delisted (Withdrawn - Terminated)
- Delisted (Withdrawn (Conditional)), Delisted (Withdrawn - Right to Re-List)
- Delisted (Temporarily No Showings), Delisted (Temporarily Off-Market)
- Delisted (Expired), Delisted (Cancelled)

## Source Updates

**Property:** Added Crexi.com, updated Loopnet.com, Remax.ca, Showcase.com, Redfin
