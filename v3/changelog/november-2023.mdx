## 31 New Property Schema Fields

### MLS Data Fields

New fields from a comprehensive MLS data feed:

- `appliances` — Included appliances list
- `assessedValues` — With `improvementsAmount`, `landAmount`, `totalAmount`, `year`
- `categories` — Category keywords from various sources
- `estimatedPrices` — Price estimates with `amountMax`, `amountMin`, `pricePerSquareFoot`
- `geoQuality` — Quality of property geoLocation data
- `hvacTypes` — HVAC system type/components
- `price.status` — Property status at time of recorded price
- `roofing` — Roof materials
- `subdivision` — Subdivision/neighborhood information

### Parcel Number as Top-Level Field

`parcelNumbers` now available as a top-level field with `number`, `year`, and `account` sub-fields.

### New Commercial Data Feed

- `architecturalStyles`, `ownerOccupied`, `topographyCode`, `zoning`
- `msaCode`, `msaName`, `cbsaCode`, `cbsaName`

### Refinance Data

- `taxExemptions`, `title`
- `transactions.documentNumber`, `transactions.transactionID`
- `trustDescription`

## Bug Fixes

- Standardized all `sqft` to `sq ft` in unit values
- Converted all emails to lowercase
- Status and price cleanup for `firstDateSeen` and `lastDateSeen` accuracy
