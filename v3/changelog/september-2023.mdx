## Property Data Updates

- Added BrightMLShomes.com as a new property source
- Added county tax record sites for Denver, Hartford, and Boulder counties
- Fixed property data transactions field duplication bug

## People Data Updates

New people data source — property owners found in the transactions field are now cross-referenced and added to the people database, enabling you to search for people in property data and vice versa.

## Source Updates

**Property:** Showcase.com, Homefinder.com, Loopnet.com, Apartments.com, Airbnb, Homes.com

**Business:** Tripadvisor.com, Opentable
