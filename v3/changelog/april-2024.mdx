## New Property Data Counties

Added county assessor data providing homeowner info, tax information, and property type codes for:

- **Nevada** — Clark County
- **Illinois** — Cook County
- **California** — Savills County
- **Pennsylvania** — Philadelphia County

## Bug Fix: Delisted Properties

Corrected an issue where a small number of delisted residential properties were incorrectly showing as "For Sale." All records have been updated to the correct status.
