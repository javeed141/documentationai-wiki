# New JSON Code Examples Added

We've updated our Getting Started documentation to included code examples when working with our product data in JSON format.

A key thing to note when downloading JSON data from Datafiniti is that the result files won't technically be JSON files. Instead, they're text files, consisting of multiple JSON records separated by newlines.

Because of this distinction, the guides include code examples for parsing this data.

Check out our guides to see how you can leverage the JSON format with product data: [ Getting Started with Product Data](https://datafiniti-api.readme.io/docs/getting-started-with-product-data)
