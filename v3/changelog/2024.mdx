# 2024 Data Release Log

<Update label="November 2024" description="Autotrace Launch & County Expansion" tags={["New Feature"]}>
### Autotrace — New Property & People Data Feature

Seamlessly combine People Data and Property Data in a single API call using the `auto_trace` parameter with a specialized property record view. [Learn more](https://docs.datafiniti.co/docs/autotrace).

### Newly Added County Assessor Data

- **Florida** — Lee County
- **Georgia** — Dekalb County
- **New Jersey** — Hudson County, Bergen County, Middlesex County

### Improved APN / Parcel Number Data

Updated how APN/Parcel numbers are imported. Users can now query the `parcelNumbers` field directly with greater accuracy, including a new `year` subfield for date filtering.

### New People Data Portal Design

Updated people data portal design matching the property data page format — auto search names, easy-to-use filters, and one-click common searches.
</Update>

<Update label="September 2024" description="Realtor License Types & Portal Search" tags={["New Feature"]}>
### Newly Added Realtor License Types

Query and identify realtor information based on state realty licenses:

- Delaware (DERED), Illinois (IDFPR), Vermont (VTREC), Wyoming (WYREC), Virginia (VADPOR)
- New Jersey (NJDOBI), Hawaii (HDCCA), Oregon (OREA), Maryland (MDMREC), Wisconsin (WIDSPS)

### Portal Improvements

- **Keyword Search** — New keyword search functionality added to property data filter list. Search via feature key, feature value, or descriptions.
- **Sequential Downloads** — "Download All" button now downloads one file at a time to reduce browser clutter.
</Update>

<Update label="July 2024" description="1M+ New Properties & Rental Price Field" tags={["New Feature", "Schema Change"]}>
### Newly Added: Our Largest Property Data Sources

Multiple brand-new property data sources added, resulting in over 1 million new properties for greater coverage of daily market trends, changes, and analytical data.

### Newly Added Realtor License Types

- Connecticut (CTREC), Florida (FREC), Kentucky (KREAB), Louisiana (LREC), Utah (UTRED)
- Oklahoma (OREC), New York (NYDOS), Arkansas (AREC), Nevada (NVRED), Kansas (KREC)
- Ohio (ODREPL), West Virginia (WVREC), Maine (MEREC)

### New Property Data Field: `mostRecentRentalPriceAmount`

Separates all most recent rental prices from other status prices. Streamlines the process of parsing through the prices field for rental prices.

### New Datafiniti Sign Up Page

Streamlined sign-up process allowing selection of free, monthly, or annual plans.

### Bug Fixes

Added more resources to handle larger daily volume of rental data for more consistent and reliable imports.
</Update>

<Update label="April 2024" description="County Assessor Expansion" tags={["New Feature", "Bug Fix"]}>

- Cook County, Illinois

### Bug Fix: Residential Delisted Status

</Update>
