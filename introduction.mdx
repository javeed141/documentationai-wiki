## Overview

Dafiniti-Javeed provides a powerful documentation platform where you can organize, manage, and publish all your project documentation effortlessly. Build comprehensive guides, API references, and tutorials using modern MDX components that enhance readability and user experience. Whether you maintain open-source projects or internal knowledge bases, Dafiniti-Javeed scales with your needs.

You get full control over content structure, themes, and interactivity without managing complex infrastructure. Start writing immediately with intuitive tools and rich previews.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at https://dashboard.example.com/register.
  </Step>
  <Step title="Connect Repository" icon="git-branch">
    Link your GitHub repo containing MDX files.
  </Step>
  <Step title="Customize Theme" icon="palette">
    Edit `theme.config.js` with your brand color `#3B82F6`.

    <CodeGroup tabs="JavaScript,TypeScript">
    ```javascript
    // theme.config.js
    export default {
      color: '#3B82F6',
      logo: '/logo.svg'
    };
    ```
    ```typescript
    // theme.config.ts
    const config = {
      color: '#3B82F6',
      logo: '/logo.svg'
    } as const;
    export default config;
    ```
    </CodeGroup>
  </Step>
  <Step title="Deploy" icon="rocket">
    Push changes to trigger a build. Access at `https://docs.example.com`.
  </Step>
</Steps>

<Callout kind="tip">
  Test locally with `npm run dev` before deploying.
</Callout>
