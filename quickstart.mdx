<Callout kind="info" title="Prerequisites">
Before starting, ensure you have:
- A modern web browser (Chrome, Firefox, or Safari)
- A valid email address for account verification
- Optional: Node.js `{>=18}` if using the CLI for advanced uploads
</Callout>

## Create Your Account and Log In

Follow these steps to set up your Dafiniti-Javeed account.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://app.dafiniti-javeed.com/signup` and enter your email, name, and password.
    
    Verify your email by clicking the link sent to your inbox.
  </Step>
  
  <Step title="Log In" icon="log-in">
    Navigate to `https://app.dafiniti-javeed.com/login`.
    
    Enter your credentials to access the dashboard.
  </Step>
</Steps>

## Set Up Your Initial Workspace

Your workspace is where you manage documentation projects.

<Steps>
  <Step title="Create Workspace" icon="folder-plus">
    From the dashboard, click **New Workspace**.
    
    Name it (e.g., `My Project Docs`) and select a template like "Starter Documentation".
  </Step>
  
  <Step title="Configure Settings" icon="settings">
    In workspace settings, set your brand color to `#3B82F6` and add your domain if needed.
  </Step>
</Steps>

## Create Your First Documentation Project

Start building your docs structure.

<Steps>
  <Step title="New Project" icon="plus">
    Inside your workspace, select **New Project** > **Documentation Site**.
    
    Choose a name like `User Guide` and pick MDX as the format.
  </Step>
  
  <Step title="Add Pages" icon="file-plus">
    Create pages via **Add Page**: `/introduction.mdx`, `/quickstart.mdx`.
  </Step>
</Steps>

## Upload and Organize Documents

Upload files and structure your content.

<Tabs>
  <Tab title="Web Upload" icon="upload">
    Drag and drop Markdown, MDX, or image files into the project tree.
    
    Use the organizer to create folders like `guides/`, `api/`.
    
    Preview changes live in the editor.
  </Tab>
  
  <Tab title="CLI Upload" icon="terminal">
    Install the CLI:
    
````bash
npm install -g @dafiniti-javeed/cli
````
    
    Authenticate and upload:
    
````bash
dafiniti-javeed login --email your@email.com
dafiniti-javeed upload --project "User Guide" ./docs/
````
  </Tab>
</Tabs>

<Callout kind="tip">
Organize early: Use folders for `concepts/`, `guides/`, and `reference/` to keep your docs scalable.
</Callout>

<Expandable title="Troubleshooting Common Issues" default-open="false">
  
### Login Fails
Clear browser cache or try incognito mode.
  
### Upload Errors
Check file sizes `{<10MB}` and ensure valid MDX syntax.
  
```javascript
// Example valid MDX structure
---
title: My Page
---
## Content
```
