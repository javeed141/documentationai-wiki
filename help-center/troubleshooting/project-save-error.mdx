---
title: Cannot Save Project
description: Fix issues saving projects in dafiniti-javeed.
---

## Symptoms

- Save button is disabled
- Error: "Unable to save project"

## Quick Fix

<Steps>
  <Step title="Check required fields">
    Ensure all required project fields are filled.
  </Step>
  <Step title="Refresh and retry">
    Reload the page and try saving again.
  </Step>
  <Step title="Clear browser cache">
    Clear cache or try a private window.
  </Step>
</Steps>
