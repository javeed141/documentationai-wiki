# V4 is here

We're incredibly excited to announce the arrival of our V4 API. Our new API provides a ton of exciting improvements:

## 1. More control over what data is generated

[Create your own views](https://developer.datafiniti.co/v4/reference#create-view) to specify which fields are returned in a search or download. You can also specify views on the fly while creating searches or downloads.

## 2. Simpler download process

We merged the old `/requests` and `/results` endpoints into just [`/downloads`](https://developer.datafiniti.co/v4/reference#get-a-download), which means you can check the download status and get the result files at the same time.

## 3. Improved tracking of record usage

A big complaint with the V3 API was credits not getting refreshed. This has been fixed and will no longer be a problem.

## 4. On-demand records

You're no longer constrained by a record cap. Use as many records as you like, so long as you're ok with paying a slightly higher on-demand price for extra records.

## 5. Better internal tools

While you won't see these, the V4 API provides our customer success teams with better tools to monitor your API usage and diagnose any issues that may come up.

## 6. Faster setup

Instantly download a custom [Postman collection](https://developer.datafiniti.co/v4/reference#postman) for your account and get running with Datafiniti right away.

## 7. Better security

We've updated our authorization protocol to provide improved security for your account.
