# 2022 Data Release Log

<Update label="December 2022" description="Commercial Property & Brand Normalization" tags={["New Feature", "Schema Change"]}>
### New Property Sources: Commercial

Added CBRE.ca and CBRE.com focused on commercial properties — over 20,000 new records.

### New Product Field: `canonicalBrand`

New top-level field for normalized brand names from scraped source data. The `brand` field now stores the raw scraped text.

### Enrichment API Improvements

Improved enrichment API across all data types — **4.5x speed increase** for enrichment calls. New logic to improve product brand matching upon import and validation.

### Fixes

- Cleaned ~100,000 property records with incorrect normalization causing duplicates in searches
- Fixed Zoopla.co.uk crawler logic where HTML changes caused scraping failures
</Update>

<Update label="November 2022" description="Broker License Numbers & Portal Filters" tags={["New Feature", "Schema Change"]}>
### New Field: `licenseNumber`

Added capability to obtain realtor/property agent license numbers (when obtainable from source).

### Property Source Updates

Updated Redfin.com and Compass.com crawl patterns for increased broker data coverage.

### New Business Source

allmenus.com added focused on delivery and fast-service restaurants. Added Amazon.co.uk and Amazon.ca.

### Portal & API Updates

- **Country & Province Dropdown Filters** — New province dropdown based on selected country in Portal Filter Builder
- **Data Enrichment Feedback** — Requesting user feedback to improve enrichment capabilities

### Fixes

- Fixed views not showing for Bronze users
- Patched property records street abbreviation issue (identified Oct 18th, patched Oct 21st) — optimized normalization and merge logic
</Update>

<Update label="June 2022" description="mostRecent Product Fields & Broker Data" tags={["New Feature", "Schema Change"]}>
### New Product `mostRecent` Fields

Added per user request to provide the most recent data from the latest scrape of a source:

- `mostRecentPriceAmount`, `mostRecentPriceAvailability`, `mostRecentPriceByDomain`
- `mostRecentPriceColor`, `mostRecentPriceCondition`, `mostRecentPriceCurrency`
- `mostRecentPriceDate`, `mostRecentPriceDomain`, `mostRecentPriceIsSale`
- `mostRecentPriceSize`, `mostRecentPriceSourceURL`

These fields work with Targeted Update to allow users to gather the most up-to-date data.

### New Sources

- 2 Property sources — UK market
- 4 Product sources — Hardware and consumer goods

### Fixes

- Fixed property bug importing `isSale` and `statuses` incorrectly
</Update>
