Dramatic impact on the property database — a significant portion (49.7 million records) was removed due to specific sources making changes. Looking at additional sources to supplement the loss going forward.

### Source Updates

Updated 2 Business, 2 Product, 3 Property sources.
</Update>

### New Sources

- 6 Product data sources

### Fixes

- Removed miscellaneous features data to improve property cluster performance
