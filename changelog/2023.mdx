<Update label="2023-11" description="DR4 & DR5 — 31 New Property Fields" tags={["schema changes", "new sources", "fixes"]}>
  Datafiniti project releases DR4 and DR5 now live. DR4 brings 3 new data sources (MLS, refinancing, commercial) with 31 new property fields. DR5 standardizes field values and elevates features into top-level fields.

  **Record Counts:** Business 132.3M (+589K new) · People 15.6M (+10.2M new) · Product 515M (+6.5M new) · Property 216.8M (+8.7M new)

  **New Property Schema Fields (31 total):**
  - **MLS Data:** `appliances`, `assessedValues` (with `improvementsAmount`, `landAmount`, `totalAmount`, `year`), `categories`, `estimatedPrices` (with `amountMax`, `amountMin`, `currency`, `dateSeen`, `pricePerSquareFoot`), `geoQuality`, `hvacTypes`, `price.status`, `roofing`, `subdivision`
  - **Parcel Data:** `parcelNumbers` (with `number`, `year`, `account`)
  - **Commercial Data:** `architecturalStyles`, `ownerOccupied`, `topographyCode`, `zoning`, `msaCode`, `msaName`, `cbsaCode`, `cbsaName`
  - **Refinance Data:** `taxExemptions`, `title`, `transactions.documentNumber`, `transactions.transactionID`, `trustDescription`

  **Bug Fixes:**
  - Converted all "sqft" to "sq ft" in unit values
  - Lowercased all emails; removed agent names with "non" from brokers
  - Added max limit to prices for price range functionality
  - Status and prices cleanup — ensured `firstDateSeen`/`lastDateSeen` filled out and top-level most recent values match arrays
  - Transactions now create status objects; converted "Rental Unit" to "Rental"

  **Source Updates (23 crawlers):**
  - Added Foreclosure.com, Land.com, Renthop.com, LA County tax records, California DRE people source
  - Changed BrightMLS crawler to new domain Nestfully.com
  - Rebuilt Apartmentguide.com
  - Updated Remax.ca, Movoto, Estately, Homes.com, Realestatebook, Har.com, Target.com, Loopnet, Century21, Apartments.com, Redfin, Homefinder, Realtytrac, Realestate.com.au, Idealista, Best Buy
</Update>
