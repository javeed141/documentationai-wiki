### New People Data Fields: Profile Images

We've introduced two new fields to the People Data schema for professional profile enrichment:

- `profileImageURLs` — Professional profile image URL(s) associated with a person

These fields are especially useful for lead enrichment workflows, recruiting and talent analytics, identity resolution models, and AI-driven profile scoring applications.

<Update label="February 2026" description="Property Data Schema Update" tags={["New Feature", "Deprecation"]}>
### New Field: `statusHistory`

We've introduced a new field, **`statusHistory`**, to the Property Data schema. This field provides a clear, chronological view of a property's status over time, allowing you to track transitions such as *for-sale listings* and *off-market events* with greater precision and context.

Each `statusHistory` entry includes:
- **Date** — The date the status change occurred
- **DateSeen** — The date Datafiniti observed the status change
- **sourceURLs** — The source URL associated with the status
- **Type** — The type of sales listing associated with the status

### Deprecation Notice: Legacy `statuses` Fields

As part of this improvement, we are beginning the deprecation of legacy `statuses` fields in favor of **`statusHistory`**.

- Existing fields will continue to work for now
- We strongly recommend updating your workflows to use `statusHistory`
- Future schema updates will prioritize `statusHistory` as the source of truth
