<Update label="March 2026" description="Profile Image Fields & Developer Use-Cases" tags={["New Feature", "Schema Update"]}>

### New People Data Fields

We've introduced two new fields to the People Data schema designed to enhance professional profile enrichment capabilities. These fields provide access to professional profile images associated with individuals.

| Field | Description |
|-------|-------------|
| `profileImageURLs` | Professional profile image URL(s) associated with a person |
| `mostRecentProfileImageURL` | The most recently observed professional profile image URL |

<CodeGroup tabs="JSON">

```json
{
  "people": [
    {
      "firstName": "Jordan",
      "lastName": "Matthews",
      "jobTitle": "Director of Operations",
      "companyName": "Example Logistics Group",
      "linkedinURL": "https://linkedin.com/in/jordan-matthews",
      "profileImageURLs": [
        "https://cdn.example.com/images/jordan-2024.jpg",
        "https://cdn.example.com/images/jordan-2023.jpg"
      ],
      "mostRecentProfileImageURL": "https://cdn.example.com/images/jordan-2024.jpg"
    }
  ]
}
```

</CodeGroup>

This enhancement is especially useful for customers building:

- Lead enrichment workflows
- Recruiting and talent analytics systems
- Identity resolution models
- AI-driven profile summarization and scoring applications

### New Field: `statusHistory`

We've introduced **`statusHistory`** to the Property Data schema — a clear, chronological view of a property's status over time. This allows you to track transitions such as *for-sale listings* and *off-market events* with greater precision.

Each `statusHistory` entry includes:

| Sub-field | Description |
|-----------|-------------|
| **Date** | The date the status change occurred |
| **DateSeen** | The date Datafiniti observed the status change |
| **sourceURLs** | The source URL associated with the status |
| **Type** | The type of sales listing associated with the status |

This enhancement is especially valuable for market trend analyses, time-based property lifecycle models, and investment or sales intelligence workflows.

View the updated schema: [Property Data Schema](https://docs.datafiniti.co/docs/property-data-schema)

<Callout kind="alert">

**Deprecation Notice:** We are beginning the deprecation of legacy `statuses` fields in favor of `statusHistory`.

- The previous `statuses` structure needed more reliable date tracking and historical context
- The split of rental statuses and sale/off-market statuses left the `statuses` field obsolete
- `statusHistory` provides a single, unified, and future-proof approach

Existing fields will continue to work for now. We'll provide ample notice before any breaking changes occur.

</Callout>

</Update>
