## New Supported Property Statuses

- For Rent By Owner (`forrentbyowner`)
- For Sale By Owner (`forsalebyowner`)

## New Property Data Schema Fields

- `companies.dateSeen`, `companies.email`, `companies.equity`, `companies.equityPercent`
- `companies.landline`, `companies.name`, `companies.phone`, `companies.title`
- `people.landline`

## Improved Property & Product Data Crawls

New county coverage:

- **Florida** — Lee County
- **Georgia** — Dekalb County
- **New Jersey** — Hudson County, Bergen County, Middlesex County

Updated sites: Yellowpages, Homes.com, Zillow.com

Newly added: AJMadison.com

## Portal Improvements

- **Keyword Search Functionality** — Search property data using keywords for feature keys, values, or descriptions
- **Sequential Downloads** — "Download All" button now downloads files one at a time
